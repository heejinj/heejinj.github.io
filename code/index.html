    FROST: Fast Robot Optimization and Simulation Toolkit

    FROST is an open-source project that I started at AMBER Lab from 2016 and is still under active development. The design goal of FROST is to create a unified software environment for symbolic modeling, trajectory optimization, model-based control design and simulation of hybrid dynamical systems, with particular interests in dynamic legged locomotion. The mathematical foundation of FROST is the hybrid zero dynamics and virtual constraints that were specifically designed to mathematically synthesize stable controllers for realizing dynamical motions.

    • Code available at: https://github.com/ayonga/frost-dev
    • Online documentation available at: http://ayonga.github.io/frost-dev/

    Features

    • Equipped with custom symbolic math toolbox for MATLAB using Mathematica Kernel
    • Hierarchical structures for describing general hybrid dynamical systems
    • Symbolic calculation of system dynamics, kinematics and other constraints
    • Specific supports for multibody systems described by standard URDF files
    • Automatic construction of trajectory optimization problems
    • Fast, reliable and scalable optimization algorithms
    • Hybrid zero dynamics and virtual constraints
    • Extendable to other dynamical systems, such as autonomous vehicles

    Applications

    FROST has been applied to many robotic platforms, ranging from biped robots to prosthetic legs and exoskeletons, from robot manipulators to autonomous vehicles. To date, FROST has been already extensively used by several research labs from Caltech, University of Michigan, and UC Berkeley, etc..
